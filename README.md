# Workout Tracker

App to track workouts: templates, calendar, active sessions, history, and personal bests.

## Links

- **Live app (use this on GitHub):** [https://ndono27.github.io/Workout-tracker/](https://ndono27.github.io/Workout-tracker/)
- **Local:** clone this repo and open `index.html` in your browser, or use a dev server (e.g. Five Server).
- **Overview page (one scrollable page):** [`docs/index.html`](https://github.com/ndono27/Workout-tracker/blob/main/docs/index.html) — same content as [`project-summary.html`](https://github.com/ndono27/Workout-tracker/blob/main/project-summary.html) at the repo root.

> Opening `index.html` from the GitHub **file viewer** (the `blob/.../index.html` page) does **not** run the app correctly — scripts and styles will not load. Use the **live app** link above instead.
