Workout Tracker - Project Showcase

Train Smarter, Track Progress, Stay Consistent

A scrollable, single-page highlight of your Workout Tracker app. It covers account handling, workout planning, active session logging, completed history, and personal best progression.

Key Features

  • Account create/login with per-user data.
  • Workout templates with exercise sets and supersets.
  • Calendar-based planning and day selection.
  • Active workout logging for reps and weight.
  • Completed workout archive by workout name.
  • Personal best tracking and updates.

User Workflow

  1. Create account or log in.
  2. Build workout templates.
  3. Select a day and start a saved workout.
  4. Log each set in active workout view.
  5. Complete and save session results.
  6. Review history and personal bests.

Architecture

  • State-driven rendering in JavaScript.
  • DOM updates by focused render functions.
  • Data normalization and validation helpers.
  • Date-keyed calendar entries and session records.

Tech Stack

  • HTML5 + CSS3 for structure and styling.
  • Vanilla JavaScript for app logic.
  • LocalStorage for persistence.
  • No backend required for MVP usage.

Future Enhancements

  • Cloud sync for cross-device access.
  • Secure authentication and hashed credentials.
  • Analytics charts for volume and progression.
  • Export/import for data backup and sharing.
  • PWA install and offline optimization.
