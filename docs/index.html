Workout Tracker

A simple place to keep your training organized: save the routines you repeat, drop them on the calendar when you plan to train, then walk through each set while you’re actually in the gym.

Your space

When you sign in, everything you build belongs to you—saved workouts, days you trained, and the numbers you hit along the way. Come back anytime and pick up where you left off.

Routines you’ll reuse

Name a workout, add exercises, and say how many sets you’re aiming for. Pair moves into supersets when you want to run them back-to-back. Once it feels right, save it so you’re not retyping the same plan every week.

The calendar

Flip through months, tap a day, and see what’s on deck—or what you already finished. Start a saved routine right from that day so what you log lines up with the date you actually trained.

Green days are a quiet nod to “you showed up.”

While you train

The active workout screen is built for thumbs and focus: each set gets a spot for reps, and weight if you use it. Finish when you’re done—no extra steps buried in menus.

When you tap complete

You get a moment that feels earned: how long the session took, and any lifts that edged past what you’d done before.

  • Confetti when you’re in the mood to celebrate.
  • A clear recap before you move on.

What you’ve already done

Past sessions stack under the workout name you used. Open one to see the sets you logged that day—handy when you’re asking “what did I squat last Tuesday?” without digging through notes.

Personal bests

Best reps and best weight get tracked per exercise inside each workout, so you can see where you’re creeping upward over time. It’s there for bragging rights, sure—but mostly so your next session has a target.
