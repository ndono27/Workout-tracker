Workout Tracker

Workout Tracker is a browser-based fitness app that helps users plan workouts, run active sessions, log set-by-set results, and track personal progress over time in a clean tabbed interface.

5Main app tabs
LocalSaved data
Vanilla JSCore logic

Core Features

  • User account creation and login
  • Workout template builder with exercises and supersets
  • Calendar-based workout planning and day selection
  • Active workout mode with reps/weight logging
  • Completed workout history grouped by workout name
  • Personal best tracking by workout and exercise

How It Works

  1. Create an account or log in.
  2. Build a saved workout template.
  3. Select a day and start the workout.
  4. Enter reps and optional weight for each set.
  5. Complete session and auto-save progress.
  6. Review completed sessions and personal bests.

Technical Summary

  • Frontend: HTML, CSS, JavaScript
  • State managed in memory and rendered to DOM
  • Persistence via LocalStorage keys per user/session
  • Validation for required workout and set inputs
  • Utility helpers for normalization and formatting

Strengths

  • Fast, no backend required to use
  • Simple and intuitive flow for daily tracking
  • Good separation between templates and logged sessions
  • Immediate visual feedback across tabs
  • Reusable structure for future analytics upgrades

Future Enhancements

  • Add cloud sync and secure account storage
  • Add charts for volume trends and progression
  • Add export/import options for backup
  • Add workout timer, rest timer, and streak metrics
  • Add shareable plans and coaching presets
