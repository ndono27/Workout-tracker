01

Account

What it’s for

Training data stays tied to whoever is signed in. Until you create an account or log in, the main app stays hidden—you only see the account panel.

What you actually do

  • Create account picks a username and password (minimum length on password). If that username is already taken, you’ll need a different one—or use the correct password for the existing account to get in.
  • Log in loads your saved workouts, calendar entries, completed-day markers, and personal bests.
  • After you’re in, a small person icon appears in the header: open it to log out when you’re done.
02

Workouts tab

Left side: build a new routine

  • Give the workout a name (for example “Pull day”).
  • Add exercise rows: each needs a name and how many sets you plan to do. Remove a row if you change your mind.
  • Add superset groups two or more exercises together; you can add more moves inside the block or remove the whole superset.
  • Save stores the template in your list. Empty or half-filled rows are skipped so you don’t save junk.

Right side: your saved list

  • Each card shows the workout name and a readable breakdown of exercises (and supersets).
  • Start on selected day kicks off that workout—but only if you’ve already picked a date on the Calendar tab first.
  • Delete removes the template and cleans up calendar entries that came from that exact template, so your history doesn’t lie about what you ran.
03

Calendar tab

Month strip

  • Use previous / next month to move around; the month title updates at the top.
  • Tap any day to select it. The selected day is highlighted.
  • Days where you’ve completed a workout pick up a different look so you can scan the month at a glance.

Selected day panel

  • The heading shows the human-readable date (weekday, month, day).
  • Choose a workout from the dropdown, then Start opens the same flow as “Start on selected day” from the Workouts tab.
  • Below that, cards list every session logged that day. Each card can be removed if you need to undo a mistaken entry (personal bests recalculate afterward).

You need a day selected before starting a workout from the calendar—otherwise the app nudges you to pick one first.

04

Active workout tab

What shows up

  • The title is the workout you started, with the date you tied it to.
  • Exercises appear in order; supersets are grouped in a bordered block so you know those sets belong together.
  • Every planned set has a row: reps (required before you can finish) and weight if you want it.

Actions

  • Discard drops the in-progress session without saving—use it if you opened the wrong day or wrong template.
  • Complete workout checks that reps are filled in, saves the session to that calendar day, marks the day complete, then takes you back toward the calendar flow.
05

Right after you complete

On screen

  • A modal shows how long the session ran, end to end.
  • If you beat earlier numbers, a short list calls out new personal bests by exercise; otherwise that part stays tucked away.
  • Continue closes the modal; there’s a bit of confetti when you finish—small reward for closing the loop.
06

Completed workouts tab

This is your archive: not “what you planned,” but what you actually logged, grouped by the workout name you used when you saved the session.

  • Each workout name is a collapsible row with a count of how many times you’ve finished it.
  • Open one to see each date underneath, with the full set-by-set text for that day—reps and weights as you entered them.
  • Newest dates tend to float toward the top within each group so recent work is easy to find.
07

Personal bests tab

Here the app rolls your history into “best so far” numbers, still organized by workout name so leg day stats don’t bleed into push day.

  • For each exercise it tracks things like best reps, best weight, and which set carried the heaviest load.
  • Exercises that exist in your templates show up even before you’ve logged a PR, so you can see what’s still blank.
  • Where the app has stored a best, you can reset that one exercise if you want a clean slate—without touching the rest of your log.
Open the app from the repo’s index.html when you want to train.
